Gearhead (Small Language Model)

Actively working on this one. In-house small language model (approx. 1.8M Parameters currently) designed from the ground up to assist with mobile equipment diagnostics and troubleshooting. I'm using this project to fill skill-gaps I've seen in my career as a diesel mechanic, especially among current generation junior techs and mechanics. Gearhead in its current state is helping me learn the fundamentals of model engineering, training, and implementation. Dataset is being built from scratch using a combination of text and a separate web app that records work order data (text, images, audio, video) and exports appropriately for indexing.

Gearhead Diagnostic Assistant (RAG, Fine-tuning)

Custom model built with Llama3.1:8b using Ollama (local) for expert diagnostics and troubleshooting in the equipment rental industry. My agent assumes the role of an equipment expert. This app is built on Python with Streamlit and supports save, print, and reset functions. Testing functionality using RAG pipeline for OEM manuals with ChromaDB. Testing speech-to-text and text-to-speech options. Will likely migrate from Streamlit soon.
